== Description ==
Scroll to top by one Click plugin will create a button that help you to go Back to Top of a page from Bottom by one click to your WordPress website. This plugin will allow visitors of your wordpress website to easily scroll back to the top of the page. It is compatible with all major browsers and should work with all themes, including responsive and high resolution themes.

= Usages =
* Go to your Dashboard after installation and you can see a menu named 'scroll to top'. Then click on the menu to configure as your requirements.

== Features =
  * Install and ready to use.
  * Live preview in the admin page.
  * Unlimited background color options.
  * After starting scrolling the button will be visible.
  * Scroll back to top with smooth animation by one click.
  * Lightweight and fast.
  * Button Position can be changed.
  * 15+ icons for the button.
  * Customizable corner round option.
  * Distance from side and bottom can also be customized.
  * Make your button width and height as you want.
